Spring Boot + React Full Stack Web Application

Tools Used for

1. Backend : Java Spring Boot, MySql, Spring JPA, Rest API Controller
2. Frontend : React (CRA), TailWindCSS, Axios
3. Testing : Github (extensive logging of each checkpoint), DataGrip (for database), PostMan, IntelliJ Idea Ultimate (for Java Spring application), VSCode (for React app)


This project was to learn the basics of a Full Stack Application + Github CI/CD pipelining

For every major feature, a new branch was created. Every commit logged a new addition, whether it was a POST mapping on the backend, a GET axios fetch on the frontend, a new react component or even a simple Testing of any response on the database

1. DataGrip is a very useful JetBrains application on top of IntelliJ to monitor if the changes made by a React application hosted on localhost are impacted on your mysql (or any db) instance
2. PostMan to validate all the HTTP requests on the backend server as always
3. Axios to implement the API calls on the Frontend after successful validation
4. TailWindCSS + additional plugins in VSCode making it very productive to create components and webpages
5. A simple yet interactive React Application bolstered by react-Navigation and react-router-dom for a very smooth UI/UX experience
6. Spring Boot (my 4th Spring Project now) making it very easy to create a Java based Web application

7. And lastly, Git/Github to monitor and track all the key moments of learning. Every PR merge marks a connection between some of my previous project or a major learning curve. All commits specify important steps to follow and obey for any future projects
